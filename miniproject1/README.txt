The modules i've used for this program, that are not included
in the standard python package are:
- beautifulsoup		(scraping)
- matplotlib		(datavisualisation)
- pandas		(datahandling)
- pillow		(imageprocessing)
- tkinter		(GUI)

To install the required modules for my program, please use
pip install -r requirements.txt

If on Debian/Ubuntu, you may run into trouble with pillows ImageTk function, in this case please use "sudo apt-get install python3-pil python3-pil.imagetk".

If on Archlinux, "use sudo pacman -S python-pillow"
